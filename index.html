Nakamoto

A privacy-first blockchain with native Bitcoin bridge. Every NAK is backed 1:1 by satoshis. Built from scratch in Go.

1:1Satoshi parity (1 BTC = 100M NAK)
12sBlock time
~150kLines of Go
TorNative networking

What It Does

Bitcoin Bridge

Deposit BTC, receive NAK at 1:1 satoshi parity. No inflation, no block reward minting. NAK only exists when BTC backs it.

Privacy by Default

Tor-native seed node. No geographic tracking. End-to-end encrypted P2P messaging built into the protocol.

On-Chain Storage

1TB decentralized storage trunks with guardian nodes. Host websites on-chain via the nak:// protocol.

Smart Contracts

WASM runtime (Wazero). Deploy smart contracts that settle in satoshi-denominated NAK.

PoS Consensus

12-second blocks. 67% threshold finality. Three-tier validator structure: main chain, trunk, and guardian nodes.

Built-In Browser

Browse websites hosted on the blockchain. Software updates distributed via the chain itself.

Tester Incentive

At mainnet launch, a trustless smart contract will redirect a percentage of protocol revenue to testnet participants. The escrow code is open-source and verifiable.

  • Protocol-enforced 5% dev fund from every transaction fee
  • Automatic distribution to testnet participant addresses
  • No trust required — the contract code handles it
  • You earn a claim on real satoshis flowing through the network

Architecture

  • Fee distribution: 40% main chain validators, 30% trunk validators, 25% guardians, 5% development fund
  • Consensus: Proof of Stake, 12-second blocks, 67% threshold
  • Storage: Decentralized 1TB trunks with guardian node redundancy
  • Networking: Tor-native seed node, encrypted P2P
  • Smart contracts: WASM runtime via Wazero
  • Updates: Protocol upgrades distributed via the blockchain
